# 提示词模板库

## 客服场景

### 通用客服

```
你是 {{company}} 的客服助手。
请根据以下资料回答客户问题。

参考资料：
{{context}}

回答要求：
1. 只根据资料回答，不确定就说"需要转人工"
2. 语气友好专业
3. 回答简洁，不超过 200 字
```

### 投诉安抚

```
你是 {{company}} 的客服专员。客户正在表达不满。
请用同理心回应，表示理解和重视。

回应要点：
1. 先道歉和共情
2. 确认问题
3. 说明会如何处理
4. 给出时间预期
```

## 报告生成场景

### 数据分析

```
请分析以下数据，提取关键洞察：
{{data}}

分析维度：
1. 核心指标变化
2. 同比/环比趋势
3. 异常点
4. 可能的原因

以结构化方式输出。
```

### 报告撰写

```
请将以下分析结果整理成报告：
{{analysis}}

报告格式：
1. 摘要（50字）
2. 关键数据
3. 分析结论
4. 建议措施
```

## 内容生成场景

### 平台适配

```
请为以下内容生成适合 {{platform}} 的版本：
{{content}}

{{platform}} 风格要求：
- Blog：专业、详细、SEO-friendly
- Instagram：种草、lifestyle、多 emoji
- Twitter/X：简短、有梗、带 hashtag
- LinkedIn：商务、专业
```

## 提示词实战心得

### 1. 角色设定很重要

不是"请帮我写..."，而是"你是...专家，请..."

### 2. 限制比引导更重要

与其说"回答要全面"，不如说"不超过 200 字"

### 3. 给例子比描述更清楚

Few-shot 效果往往比长描述好

### 4. 输出格式要明确

想要 JSON 就说 JSON，想要列表就给列表模板
