# 第 0 章：5 分钟，见证 AI 应用的诞生

> 小林在 TechStore 干运营。TechStore 是个中等规模的电子产品电商——手机、电脑、配件，什么都卖。
>
> 某天午饭后，同事随口提了一句："听说有个叫 Dify 的东西，不写代码就能做 AI 应用。"
>
> 小林不太信："我连 Python 都不会。"
>
> "真不用写代码，拖来拖去就行。"
>
> 下午三点，小林带着试试看的心态打开了 Dify 官网。

## 注册账号

1. 打开浏览器，访问 [cloud.dify.ai](https://cloud.dify.ai)
2. 点击右上角的「Get Started」按钮
3. 你可以选择使用 GitHub 账号或 Google 账号登录，也可以用邮箱注册
4. 完成注册后，你会进入 Dify 的工作台界面

整个过程不到 1 分钟。

## 探索现成的应用

注册完先别急，不着急自己动手。先看看别人做了什么。

1. 点击左侧导航栏的「探索」（Explore）
2. 你会看到一个应用商店，里面有各种各样的 AI 应用模板
3. 找到「翻译助手」或「文章摘要」这类简单的应用
4. 点击「运行」，输入一段文字，看看效果

你会看到 AI 按照预设的规则处理了输入，然后给出了结果。

注意，这不是随便聊天——它是一个有特定功能的 AI 应用，和你直接跟 ChatGPT 对话是两回事。

## 看看它是怎么做的

接下来看看这个应用的"内部结构"。

1. 在应用详情页，点击「添加到工作区」
2. 回到「工作室」（Studio），你会看到这个应用出现在你的列表里
3. 点击它，进入编辑界面
4. 你会看到一个可视化的界面，显示了这个应用的"配方"——它用了什么模型、设置了什么提示词（Prompt）、定义了什么输入输出

这就是 Dify 干的事：用可视化的方式，把 AI 的能力组装成一个应用。

## 回头看看你刚才做了什么

五分钟而已。你注册了一个账号，跑了一个现成的 AI 应用，还打开看了它是怎么拼起来的。

::: tip 关键认知
AI 应用不是黑魔法。模型、提示词、输入、输出——就这几样东西拼在一起。Dify 做的事情就是把这些东西摊开给你看，让你能改、能调、能从零开始搭。
:::

小林看着屏幕，心里开始有了想法：

*"如果这些模板都能这么快跑起来，那我能不能做一个专门回答 TechStore 客户问题的机器人？"*

当然可以。30 分钟够了。
