# 第 0 章：5 分钟，见证 AI 应用的诞生

> 小林是 TechStore 的运营负责人。TechStore 是一家中型电子产品电商，卖手机、电脑、配件。
>
> 最近，小林听同事说："有个叫 Dify 的东西，能让你自己做 AI 应用。"
>
> "自己做？"小林半信半疑，"我又不会写代码。"
>
> "不用写代码，拖拖拽拽就行。"
>
> 带着好奇，小林打开了 Dify 的官网……

## 注册账号

1. 打开浏览器，访问 [cloud.dify.ai](https://cloud.dify.ai)
2. 点击右上角的「Get Started」按钮
3. 你可以选择使用 GitHub 账号或 Google 账号登录，也可以用邮箱注册
4. 完成注册后，你会进入 Dify 的工作台界面

整个过程不到 1 分钟。

## 探索现成的应用

注册完成后，先别急着创建自己的应用。让我们先看看别人做了什么。

1. 点击左侧导航栏的「探索」（Explore）
2. 你会看到一个应用商店，里面有各种各样的 AI 应用模板
3. 找到「翻译助手」或「文章摘要」这类简单的应用
4. 点击「运行」，输入一段文字，看看效果

神奇的事情发生了：AI 按照预设的规则处理了你的输入，然后给出了结果。

**这不是普通的 ChatGPT 对话，而是一个有特定功能的 AI 应用。**

## 看看它是怎么做的

现在，让我们看看这个应用的"内部结构"。

1. 在应用详情页，点击「添加到工作区」
2. 回到「工作室」（Studio），你会看到这个应用出现在你的列表里
3. 点击它，进入编辑界面
4. 你会看到一个可视化的界面，显示了这个应用的"配方"——它用了什么模型、设置了什么提示词（Prompt）、定义了什么输入输出

这就是 Dify 的核心：**用可视化的方式，把 AI 的能力组装成一个应用。**

## 你刚才做了什么？

让我们回顾一下过去 5 分钟发生的事情：

- 你注册了一个 Dify 账号
- 你运行了一个现成的 AI 应用
- 你看到了这个应用是如何被"组装"出来的

::: tip 关键认知
AI 应用不是黑魔法。它由明确的组件构成——模型、提示词、输入、输出。Dify 把这些组件可视化了，让你能够理解、修改、甚至从零创建。
:::
