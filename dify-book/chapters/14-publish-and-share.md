## 安全与权限

IT 部的安全专员找到小林：*"你这些 AI 工具，权限是怎么控制的？"*

小林把他学到的东西整理了一下：

**API Key 安全：**
- 不要把 Key 写在前端代码里
- 通过后端服务器中转调用
- 定期更换 Key

**访问控制：**
- 内部工具设置访问密码
- 通过 API 的 user 参数追踪谁在用
- 敏感应用限制 IP 白名单

**成本控制：**
- 监控 API 调用量和 Token 消耗
- 设置每日/每月调用上限
- 对内部工具用便宜的模型

**数据隐私：**
- 注意知识库里有没有敏感信息
- 公开应用不要关联内部知识库
- 如果数据很敏感，考虑私有部署

安全专员听完，点点头：*"听起来你考虑得还挺全面的。"*
